Welcome to SkyFlow Portal

This web application showcases real-time weather integration and user preferences storage using a lightweight SQL backend database.

Frontend API Integration: Uses Open-Meteo Geocoding & Weather REST APIs to fetch real-time atmospheric states without API keys.
SQL Settings DB: Your favorite city, theme preference, and temperature scales are saved securely in a backend SQLite database.
Dynamic UI: Uses CSS Custom Variables to instantly transition between Light and Dark mode, with full responsiveness built in.
